SVG Images with PNG fallback for IE 8

This demo has SVG logos with PNG fallbacks for older browsers. SVG is resolution independent so additional graphics for Retina displays aren't neccessary. The logos shown make use of sprites so that both Yale Blue and white reversed images are saved together. Additional PNG graphics are also included for completeness. See additional notes in the markup of this demo page.
